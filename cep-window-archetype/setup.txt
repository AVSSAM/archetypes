Product: Template for WSO2 CEP SIDDHI FUNCTION

STEPS:

 1. Navigate to "<SIDDHI_EXTENSION_HOME>/org.wso2.siddi.extension.archetype.cep.window-archetype" and run the following command
         mvn clean install

 2. Navigate to "<SIDDHI_FUNCTION_HOME>" and run the following command to create the Siddhi function
    mvn archetype:generate
        -DarchetypeGroupId=org.wso2.carbon.extension.archetype
        -DarchetypeArtifactId=org.wso2.siddi.extension.archetype.cep.window-archetype
        -DarchetypeVersion=1.0.2-SNAPSHOT
        -DgroupId=org.wso2.siddi.extension.window
        -DartifactId=org.wso2.siddi.extension.window.sample
        -Dversion=1.0.0
        
    Please enter the <siddi_window_name> in lower camel case in above DartifactId command line
    eg:- org.wso2.siddi.extension.window.sample

 3. Enter the authenticator name after executing steps 1 & 2. Please enter the Function name in upper camel case
    eg:- siddi_window_name : : Sample

 4. Confirm the connector name
