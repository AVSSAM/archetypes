Product: Template for WSO2 SIDDHI-EXTENSION-STREAM_FUNCTION

STEPS:

 1. Navigate to "<SIDDHI-EXTENSION-STREAM_FUNCTION_HOME>/siddhi-streamProcessor-archetype" and run the following command
         mvn clean install

 2. Navigate to "<SIDDHI-EXTENSION-STREAM_FUNCTION_HOME>" and run the following command to create the siddhi streamfunction
    mvn archetype:generate
        -DarchetypeGroupId=org.wso2.carbon.extension.archetype
        -DarchetypeArtifactId=org.wso2.siddi.extension.archetype.streamFunction-archetype
        -DarchetypeVersion=1.0.2-SNAPSHOT
        -DgroupId=org.wso2.siddi.extension.streamfunction
        -DartifactId=org.wso2.siddi.extension.streamfunction.sample
        -Dversion=1.0.0
        
    Please enter the <siddi_stream_function_name> in lower camel case in above DartifactId command line
    eg:- org.wso2.siddi.extension.streamfunction.sample

 3. Enter the authenticator name after executing steps 1 & 2. Please enter the Function name in upper camel case
    eg:- siddi_stream_function_name : : Sample

 4. Confirm the connector name
