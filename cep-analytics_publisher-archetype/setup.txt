Product: Template for WSO2 CEP SIDDHI FUNCTION

STEPS:

 1. Navigate to "<SIDDHI_FUNCTION_HOME>/org.wso2.carbon.authenticator.connector-archetype" and run the following command
         mvn clean install

 2. Navigate to "<SIDDHI_FUNCTION_HOME>" and run the following command to create the Siddhi function
    mvn archetype:generate
        -DarchetypeGroupId=org.wso2.carbon.extension.archetype
        -DarchetypeArtifactId=org.wso2.carbon.extension.archetype.cep.analytics_publisher-archetype
        -DarchetypeVersion=1.0.2-SNAPSHOT
        -DgroupId=org.wso2.carbon.extension.analytics_publisher
        -DartifactId=org.wso2.carbon.extension.analytics_publisher.sample
        -Dversion=1.0.0
        
    Please enter the <siddi_stream_processor_name> in lower camel case in above DartifactId command line
    eg:- org.wso2.carbon.extension.analytics_publisher.sample

 3. Enter the authenticator name after executing steps 1 & 2. Please enter the Function name in upper camel case
    eg:- carbon_analytics_publisher_name : : Sample

 4. Confirm the connector name
