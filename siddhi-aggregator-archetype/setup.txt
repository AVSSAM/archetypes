Product: Template for WSO2 CEP SIDDHI-EXTENSION-AGGREGATOR

STEPS:

 1. Navigate to "<SIDDHI_AGGREGATOR_HOME>/siddhi-aggregator-archetype" and run the following command
         mvn clean install

 2. Navigate to "<SIDDHI_AGGREGATOR_HOME>" and run the following command to create the siddhi aggregator
    mvn archetype:generate
        -DarchetypeGroupId=org.wso2.carbon.extension.archetype
        -DarchetypeArtifactId=org.wso2.siddi.extension.archetype.aggregator-archetype
        -DarchetypeVersion=1.0.2-SNAPSHOT
        -DgroupId=org.wso2.siddi.extension.aggregation
        -DartifactId=org.wso2.siddi.extension.aggregation.sample
        -Dversion=1.0.0
        
    Please enter the <siddi_aggregator_name> in lower camel case in above DartifactId command line
    eg:- org.wso2.siddi.extension.aggregation.sample

 3. Enter the authenticator name after executing steps 1 & 2. Please enter the Function name in upper camel case
    eg:- siddi_aggregator_name : : Sample

 4. Confirm the connector name
