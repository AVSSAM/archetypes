Product: Template for WSO2 SIDDHI-EXTENSION-STREAM_PROCESSOR

STEPS:

 1. Navigate to "<SIDDHI-EXTENSION-STREAM_PROCESSOR>/siddhi-streamProcessor-archetype" and run the following command
         mvn clean install

 2. Navigate to "<SIDDHI-EXTENSION-STREAM_PROCESSOR>" and run the following command to create the siddhi streamprocessor
    mvn archetype:generate
        -DarchetypeGroupId=org.wso2.carbon.extension.archetype
        -DarchetypeArtifactId=org.wso2.siddi.extension.archetype.streamProcessor-archetype
        -DarchetypeVersion=1.0.2-SNAPSHOT
        -DgroupId=org.wso2.siddi.extension.streamprocessor
        -DartifactId=org.wso2.siddi.extension.streamprocessor.sample
        -Dversion=1.0.0
        
    Please enter the <siddi_stream_processor_name> in lower camel case in above DartifactId command line
    eg:- org.wso2.siddi.extension.streamprocessor.sample

 3. Enter the authenticator name after executing steps 1 & 2. Please enter the Function name in upper camel case
    eg:- siddi_stream_processor_name : : Sample

 4. Confirm the connector name
     Y : : Y
