Product: Template for WSO2 ANALYTICS-EVENT-PUBLISHER

STEPS:

 1. Navigate to "<ANALYTICS-EVENT-PUBLISHER_HOME>/analytics-event-publisher-archetype" and run the following command
         mvn clean install

 2. Navigate to "<ANALYTICS-EVENT-PUBLISHER_HOME>" and run the following command to create the carbon adaptor
    mvn archetype:generate
        -DarchetypeGroupId=org.wso2.carbon.extension.archetype
        -DarchetypeArtifactId=org.wso2.carbon.extension.archetype.analytics_publisher-archetype
        -DarchetypeVersion=1.0.2-SNAPSHOT
        -DgroupId=org.wso2.carbon.extension.analytics_publisher
        -DartifactId=org.wso2.carbon.extension.analytics_publisher.sample
        -Dversion=1.0.0
        
    Please enter the <carbon_stream_processor_name> in lower camel case in above DartifactId command line
    eg:- org.wso2.carbon.extension.analytics_publisher.sample

 3. Enter the authenticator name after executing steps 1 & 2. Please enter the Function name in upper camel case
    eg:- carbon_analytics_publisher_name : : Sample

 4. Confirm the connector name
