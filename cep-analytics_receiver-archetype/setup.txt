Product: Template for WSO2 CEP SIDDHI FUNCTION

STEPS:

 1. Navigate to "<CARBON_ADAPTOR_HOME>/cep-analytics_receiver-archetype" and run the following command
         mvn clean install

 2. Navigate to "<CARBON_ADAPTOR_HOME>" and run the following command to create the Carbon adaptor
    mvn archetype:generate
        -DarchetypeGroupId=org.wso2.carbon.extension.archetype
        -DarchetypeArtifactId=org.wso2.carbon.extension.archetype.cep.analytics_receiver-archetype
        -DarchetypeVersion=1.0.2-SNAPSHOT
        -DgroupId=org.wso2.carbon.extension.analytics_receiver
        -DartifactId=org.wso2.carbon.extension.analytics_receiver.sample
        -Dversion=1.0.0
        
    Please enter the <carbon_stream_receiver_name> in lower camel case in above DartifactId command line
    eg:- org.wso2.carbon.extension.analytics_receiver.sample

 3. Enter the authenticator name after executing steps 1 & 2. Please enter the Function name in upper camel case
    eg:- carbon_analytics_receiver_name : : Sample
